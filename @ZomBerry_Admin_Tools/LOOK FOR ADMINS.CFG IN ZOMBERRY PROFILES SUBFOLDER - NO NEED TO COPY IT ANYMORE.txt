LOOK FOR ADMINS.CFG IN ZOMBERRY PROFILES SUBFOLDER - NO NEED TO COPY IT ANYMORE

1. Install the mod
2. Allow the server to start up with this mod loaded once
3. Take a look inside a folder specified in -profiles launch option
   If you don't have a -profiles folder specified:
    - if hosting on your PC (or on a dedicated server) - specify it in .bat/.ps1/.sh file (e.g. profiles=ServerProfiles)
    - if hosting using a hosting provider - 99.9% you have it specified. Take a look at RPT logs in logs manager, you will see a full list of launch options passed to server in the first few lines of the log file

The ONLY exception (if you don't use the 4NetPlayers hosting - this doesn't apply to you!): 
4NetPlayers users - you need to create an empty text file, rename it to admins.cfg (NOT admins.cfg.txt! Google "Enable file extensions in explorer"),
add your SteamID64 inside and upload it to BOTH Upload and Saves FTP servers

You won't find admins.cfg here! It was done to avoid the "I've edited admins.cfg and it doesn't work" problem - people didn't understand what profiles folder is and
were trying to edit it right here (it wasn't effective because mod can't reliably find the location of /this/ folder and as of now it doesn't even try to anymore)

Any questions? Make sure to read through instructions and FAQ, and contact us https://discord.gg/BfMZxRh :)
